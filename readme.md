#BookConv

定跡変換ツール

ShogiGUIの定跡などから各エンジンの定跡に変換できます。



## 使い方

入力ファイル、出力ファイルを設定して変換ボタンを押します。

入力ファイルの形式は拡張子でフォーマットを判断します。
出力ファイルの形式はドロップダウンで選択します。


## 対応している定跡ファイルの形式
### 入力
* ShogiGUIの定跡（.sbk)
* 技巧定跡(book.bin)
* やねうら王定跡(.db)

### 出力
* Apery定跡
*  技巧定跡
* やねうら王定跡
* ShogiGUI定跡

※技巧定跡からShogiGUIの定跡に変換した場合、評価値など失われますのでご注意ください。
